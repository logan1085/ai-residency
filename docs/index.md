## Program Shape

This repository converts the Thoughtful Labs AI Research Curriculum into a maintainable MkDocs site. Each module is designed around four visible artifacts: notes, implementation work, experiments, and a proof-of-mastery deliverable.

<div class="curriculum-grid">
  <div class="module-card">
    <h3>Build Foundations</h3>
    <p>Implement transformers, training loops, retrieval, and evaluation from first principles before relying on frameworks.</p>
  </div>
  <div class="module-card">
    <h3>Practice Research</h3>
    <p>Read papers, reproduce claims, design ablations, and write up results with evidence.</p>
  </div>
  <div class="module-card">
    <h3>Ship Systems</h3>
    <p>Package models, agents, and demos with observability, safety checks, and portfolio-grade documentation.</p>
  </div>
</div>
