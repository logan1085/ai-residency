# Module 10: Capstone

Module 10 packages the residency into a complete research artifact: a scoped question, working system, evaluation, write-up, and demo.

## Learning Objectives

- Convert an open-ended AI idea into a researchable question.
- Build a system that tests the question.
- Evaluate the system with credible evidence.
- Present the work as a portfolio-ready artifact.

## Subtopics

- Problem framing and hypothesis design.
- Project scoping and milestone planning.
- System implementation.
- Evaluation design and ablations.
- Demo design.
- Paper-style writing and final presentation.

## Assignments

- Submit a one-page capstone proposal.
- Create a milestone plan with weekly evidence checkpoints.
- Run a baseline before adding the core innovation.
- Draft the final write-up before the last implementation week.

## Projects

- A novel RAG, agent, multimodal, safety, or evaluation system.
- A paper reproduction with a meaningful extension.
- A tool that improves AI research workflow and is evaluated on real tasks.

## Proof of Mastery

- Public demo or recorded walkthrough.
- Paper-style write-up with abstract, method, results, limitations, and future work.
- Evaluation dataset and reproducible scripts.
- Final presentation with clear claims and evidence.
